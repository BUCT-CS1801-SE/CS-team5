```bash
pip install django
```

## frontend

前端，请预先安装vue,vuecli,elementui,axios,vue-router

vue版本为vue2.6.12,保证版本在2.5以上,不要装vue3

```bash
npm install vue
```

vuecli版本为4.5.13,最最好保证版本在4.5以上

```bash
npm install -g @vue/cli
```

elementui版本为2.15.1

```bash
npm i element-ui -S
```
